# AcneSafeCheck

Free acne-safe / comedogenic ingredient checker. Static site (HTML + CSS + vanilla JS), no build step.

## Pages
- index.html — Acne safe ingredient checker (home)
- comedogenic-ingredient-checker.html
- pore-clogging-ingredient-checker.html
- comedogenic-ingredients-list.html — searchable 134-ingredient database
